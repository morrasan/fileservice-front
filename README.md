# File Service - front
The simple file service for the store files

## Stack version
1. Vue 3.4.21
2. Node 19.5.0
3. Vuetify 3.5.16

## Install / run
1. git clone https://github.com/morrasan/fileservice-front.git
2. docker compose up
3. open browser http://localhost:3023

## Environment
1. port 3023
2. http://localhost:8000 - API_URL
